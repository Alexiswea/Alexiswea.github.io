Turtle Project

About the project

The project was abotu creating a graphical scene using Python's Turtle graphics module. I used the building block functions like draw_rectange, draw_circle, draw_triangle, etc to create functions that made shapes to where I could create images like a tree and clouds. Then I refactored the code by creating a code like draw_house and implementing the other functions eariler to create an image of a house so I would not have to retype the functions of draw_rectange,etc instead this helps me create a thousand houses if I so please.

Final Scene

The final scene is a house with clouds and the sun. The house uses rectangles and triangles, while the clouds are just overlapping circles.
